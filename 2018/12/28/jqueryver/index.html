jQuery 라이브러리 버전, 무슨 차이일까?

2018-12-28

jQuery 1.x / 2.x / 3.x

  • Jquery 1.x 는 Internet Explorer 8 이전 버전에서의 동작까지 모두 지원하는 버전이다.

  • Jquery 1.x 버전에서도 특히 1.9 버전은 하위 호환성이 떨어지는 버전인데, 그 이유는 1.9버전에서 없어진 것들이 많기 때문이다.

  • 이때문에 연식이 오래된 툴의 경우에는 호환성 문제 때문에 1.8버전에서 머무는 경우가 있다.

  • 이러한 문제를 해결하기 위해 1.9 버전 이상을 사용하면서 1.8 이하 버전의 호환성을 유지하려면 jQuery Migrate라는 플러그인을 사용하면 된다.

  • Jquery 2.x 는 1버전에서 지원하던 하위버전에 대한 지원을 중단한 버전이며, Internet Explorer 9 이상에서 동작한다.

  • 마지막 Jquery 3.x 는 기존 버전과의 호환성을 유지하며, 간결하게 작성되고 더욱 빠르게 동작하도록 변경되었다.

공식 jQuery 블로그 메모 인용

IE8, Opera 12, Safari 5 등 가장 다양한 브라우저를 지원해야하는 경우 jQuery-Compat 3.0.0 패키지를 사용하십시오. 모든 웹 사이트 방문자에게 최상의 호환성을 제공하므로 대부분의 웹 사이트에이 버전을 사용하는 것이 좋습니다.

웹 사이트가 항상 최신의 브라우저 용으로 제작 된 경우 또는 사용중인 브라우저 엔진을 알고있는 webview (예 : PhoneGap 또는 Cordova)에 포함 된 HTML 기반 앱인 경우 jQuery 3.0.0 패키지로 이동하십시오.

우리가 달리 알릴 때까지, 두 패키지는 모두 동일한 번호의 메이저 및 마이너 버전의 공개 API를 포함하게됩니다. 개발자가 둘 사이를 전환하고 제 3 자 jQuery 플러그인과 최대한 호환 될 수 있도록해야합니다.
