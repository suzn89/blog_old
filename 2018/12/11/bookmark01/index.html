CSS3 관련 Bookmark_01

2018-12-11

소개하기 전에

버튼에 마우스를 hover 했을때, active 했을때, 떠났을때를 구현한다는건 버튼하나지만 여간 귀찮은 일이 아닐 수 없다.
이런 편리한 툴들을 권하고 싶은 사람은, 최소한 자동으로 생성해주는 도구가 뱉어낸 CSS 아웃풋을 보고 머리속에서 이미지로 그려낼 수 있는 사람이다.
무슨 말인지는 알고는 써야하지 않을까 싶다.

CSS CHEAT SHEET


CSS CHEAT SHEET

코린이 시절에 아주 도움이 많이 되었던 사이트.
사실은 이 페이지를 통째로 A3에 인쇄해서 책상에 붙여 뒀었다.
간결하지만 기초적이고, 기본적이며 꼭 필요한 것들만 있다.

CSS Button Generator


CSS Button Generator

CSS Button Generator 같은 경우는 사용한지 꽤 된 사이트이다.
클래스 이름을 직접 지정하고, 버튼 기본스타일도 있어서 조금만 수정하면 스타일 가이드를 작성할때 아주 유용하다.

CSS-TRICKS


CSS-TRICKS

링크걸린 주소는 CSS로 여러가지 도형을 만드는 방법을 소개한 페이지이다.
둘러보면 그 외에 CSS로 할 수있는 것들을 볼 수 있다.

CSS Gradient Generator


CSS Gradient Generator

CSS로 그라데이션을 생성해주는 사이트.

dirtymarkup


dirtymarkup

처음부터 깔끔하게 마크업하면 좋지만..
사용해본 것중 가장 괜찮은 것같다. 들여쓰기도 탭으로 할것인지, 2칸들여쓸것인지, 4칸 들여쓸것인지 선택 할수 있다.
html / css / js / api 를 지원한다.

CSS 3.0 MAKER


CSS 3.0 MAKER

둥근모서리, 플렉스박스, 그라데이션, 에니메이션, 트랜지션.. 그림자등 여러가지 css3.0 을 빠른시간에 생성해준다.
간단히 만든 후에, 조금 손봐서 사용하면 유용하다.
벤더프리픽스도 함께 나온다.

유용하거나 자주 사용하는 사이트들 위주로 작성해보았다.
나도 여기저기 뒤지고, 구글링 하다가 발견한 곳들도 있지만 내가 빅데이터에 기여할수 있길 바래본다.
