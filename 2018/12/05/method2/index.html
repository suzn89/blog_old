.children()

Javascript

.children() 메서드를 이용하여 자식요소 선택가능

.next()

Javascript

.next() : 바로 다음 형제 엘리먼트 선택가능

.nextAll()

Javascript

.nextAll() : 바로 다음 형제 앨리먼트들 모두 선택가능

.nextUntil()

Javascript

HTML

.nextUntil() : 형제 엘리먼트중 ~까지(마지막 미포함)
예제 HTML에서는 2,3,4만 선택됨

.parent()

Javascript

HTML

.parent() : 부모 엘리먼트 찾기

.parents()

Javascript

.parents() : 조상 엘리먼트를 찾는다. 부모 엘리먼트 위까지!

.parentsUntil()

Javascript

.parentsUntil() : 특정한 조건이 참이 되기 전까지 계속 부모요소를 찾아낸다.

.find()

Javascript

.find() : 하위항목에서 찾는다!

.is()

Javascript

.is() : 존재 여부를 체크한다! true / false 값을 갖는다.
